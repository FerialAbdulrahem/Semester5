## ⚡ Digital System Design – Smart Parking System
**Course:** CSEN 605 – Digital System Design  
**Languages/Tools:** VHDL, FPGA Simulation  

📖 **Description:**  
Designed and implemented a **Smart Parking System** using digital logic and FPGA simulation. The project focused on hardware description, finite state machines, and real-time control for managing parking slots efficiently.  

📄 [Read Project Description (PDF)](https://github.com/FerialAbdulrahem/Semester5/blob/main/S5Projects/DSD/(_CSEN605_)%203%20-%20Project%20Description%20(Project)%20(3).pdf)  

✅ **Outcome:** Delivered a working digital design project simulating automated parking management with proper slot allocation and control logic.  
📎 [Go to Project Folder](https://github.com/FerialAbdulrahem/Semester5/tree/main/S5Projects/DSD/4_1)
